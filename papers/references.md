# Research References & Related Work

This document is a collection of research and related work in the supply-chain security, deception, and trust management space.

## Supply-Chain Security

*   **TUF (The Update Framework)**: [theupdateframework.github.io](https://theupdateframework.github.io/) — A security framework that helps software update systems become resilient against compromises.
*   **in-toto**: [in-toto.io](https://in-toto.io/) — A framework to secure the integrity of software supply chains by providing verifiable evidence of all build steps.
*   **SLSA (Supply-chain Levels for Software Artifacts)**: [slsa.dev](https://slsa.dev/) — A security framework from Google and others to improve the integrity of software supply chains.

## Deception & Honeypots

*   **Honeypots for Active Defense**: (NIST/SANS) Research into using deception and redirected traffic to detect and monitor malicious actors.
*   **Behavioral Sandboxing**: (Cuckoo Sandbox, etc.) The process of detonating untrusted binaries or scripts in isolated environments.

## Agentic Development & Vibe Coding

*   **Agentic Workflows**: (Emergent Research) The shift from human-oriented development towards autonomous AI agents (Antigravity, Cursor, etc.).
*   **The Velocity of Software Production**: Research into how AI-assisted coding is changing the speed and scale of dependency intake.
