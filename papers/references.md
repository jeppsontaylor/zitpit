# Research References & Related Work

This document is a collection of research and related work in the supply-chain security and trust management space, with deception and sandboxing listed as supporting techniques rather than the root trust model.

## Supply-Chain Security

*   **TUF (The Update Framework)**: [theupdateframework.github.io](https://theupdateframework.github.io/) — A security framework that helps software update systems become resilient against compromises.
*   **in-toto**: [in-toto.io](https://in-toto.io/) — A framework to secure the integrity of software supply chains by providing verifiable evidence of all build steps.
*   **SLSA (Supply-chain Levels for Software Artifacts)**: [slsa.dev](https://slsa.dev/) — A security framework from Google and others to improve the integrity of software supply chains.

## Agentic Development & Packaging Surfaces

*   **Anthropic Claude Code Getting Started**: [docs.anthropic.com/en/docs/claude-code/getting-started](https://docs.anthropic.com/en/docs/claude-code/getting-started) — Official documentation showing npm-based installation flows.
*   **Anthropic Claude Code Security**: [docs.anthropic.com/en/docs/claude-code/security](https://docs.anthropic.com/en/docs/claude-code/security) — Official documentation noting that allowed MCP servers are configured in source control.
*   **Anthropic Claude Code MCP**: [docs.anthropic.com/en/docs/claude-code/mcp](https://docs.anthropic.com/en/docs/claude-code/mcp) — Official documentation on MCP tool access and local agent tooling.
*   **Anthropic Claude Code Devcontainers**: [docs.anthropic.com/en/docs/claude-code/devcontainer](https://docs.anthropic.com/en/docs/claude-code/devcontainer) — Official documentation on devcontainer isolation and outbound access controls.
*   **GitHub Security Lab Advisories**: [securitylab.github.com/advisories](https://securitylab.github.com/advisories/) — Official advisories and workflow-leak research across open-source projects.
*   **axios Security Overview**: [github.com/axios/axios/security](https://github.com/axios/axios/security) — Official project security page and advisory history for a widely used npm package.

## Deception & Honeypots

*   **Honeypots for Active Defense**: (NIST/SANS) Research into using deception and redirected traffic to detect and monitor malicious actors.
*   **Behavioral Sandboxing**: (Cuckoo Sandbox, etc.) The process of detonating untrusted binaries or scripts in isolated environments.

## Agentic Development & Vibe Coding

*   **Agentic Workflows**: (Emergent Research) The shift from human-oriented development towards autonomous AI agents (Antigravity, Cursor, etc.).
*   **The Velocity of Software Production**: Research into how AI-assisted coding is changing the speed and scale of dependency intake.
